# AI TYCOON 🤖

AI 스타트업을 경영하는 브라우저 기반 경영 시뮬레이션 게임입니다.

## 소개

플레이어는 AI 스타트업의 대표가 되어 회사를 성장시킵니다. 직원을 채용하고, 프로젝트를 진행하며, 연구소에서 기술을 연구하고, 업그레이드를 통해 회사를 발전시켜 나갑니다.

## 주요 기능

- 💰 **자금 관리** — 회사의 자금과 회사 가치를 추적
- 🏢 **회사 성장** — 회사 레벨업 및 명성 시스템
- 👥 **직원 채용** — 사무실에 직원을 채용하고 관리
- 📁 **프로젝트 진행** — 다양한 프로젝트를 수행하여 성장
- 🔬 **연구소** — 기술 연구를 통한 역량 강화
- 🔧 **업그레이드** — 회사 시설 및 능력 업그레이드
- 📰 **실시간 뉴스 티커** — 게임 내 이벤트 알림

## 기술 스택

- HTML / CSS / JavaScript (Vanilla)
- Firebase Realtime Database (데이터 저장)

## 파일 구조

```
├── index.html          # 메인 페이지
├── style.css           # 스타일시트
├── game.js             # 게임 로직
└── firebase-config.js  # Firebase 설정
```

## 실행 방법

`index.html` 파일을 브라우저에서 열면 바로 플레이할 수 있습니다.
